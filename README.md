# Algorithm Visualizer 👀🔢

This project was created to learn the basics of Vue. All basic constructs like ref or interaction between DOM and script are implemented.

## How to start? 🚩

Just download the project and navigate into the folder type `npm i`, after that just type `npm run dev` and a local server would be loaded. You can now order the arrays!
